# /summarize-video

인자로 전달된 유튜브 영상 URL의 자막을 추출하고 한국 증시 관점에서 핵심 내용을 요약한다.

## 사용법

```
/summarize-video <youtube_url>
```

## 실행 절차

1. 인자에서 유튜브 URL 추출

2. `mmk youtube transcript <url>` 실행하여 자막 텍스트 수집
   - 실패 시 (자막 없는 영상): "자막 없음" 메시지와 함께 종료
   - 자막이 너무 짧으면 (100자 미만): "내용 부족" 으로 처리

3. 자막 텍스트를 분석하여 다음 항목으로 구성된 한국어 요약 생성:

   **요약 작성 기준:**
   - 전체 길이: 400~600자 이내
   - 객관적이고 중립적인 톤
   - 한국 증시 투자자에게 유용한 정보 중심

   **포함할 항목:**
   - 📌 핵심 내용: 오늘 증시 주요 이슈 (2~3줄)
   - 📈 주요 종목/섹터: 언급된 종목, 섹터, 지수 (없으면 생략)
   - 💬 전문가 의견: 핵심 전망이나 분석 (있는 경우)
   - ⚠️ 주의사항: 리스크 요인 (있는 경우)

4. 결과를 다음 JSON 형식으로 출력:
   ```json
   {
     "summary": "요약 텍스트",
     "keywords": ["증시", "시황", "코스피"],
     "mentioned_stocks": ["삼성전자", "SK하이닉스"],
     "sentiment": "긍정/중립/부정"
   }
   ```

## 주의사항
- 자막이 없거나 자막 추출에 실패하면 JSON 대신 오류 메시지 출력
- 요약은 반드시 자막 내용에 기반해야 함 (추론/추가 정보 금지)
- 투자 권유나 매수/매도 추천 문구 포함 금지
